### 🚀 Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Rendering Engine**: Native Vanilla WebGL/HTML5 `<canvas>`

### 💎 Key Features
- **Cinematic Scrubbing Engine**: 120-frame high-resolution WebP sequence preloaded instantly into the client's memory. Synced directly to `framer-motion` scroll data with strict `requestAnimationFrame` hooks — bypassing React’s laggy render cycles entirely.
- **Glassmorphic UI**: High-end Apple-style floating navbars and translucent gradient cards leveraging `backdrop-blur-xl`.
- **Responsive Parallax Typography**: Fully-responsive floating overlay typography that shrinks, moves, and tracks the user flawlessly without overlapping. 
- **Film Grain & Vignette**: A beautiful CSS-only overlay combining radial gradient vignetting with a stitched SVG turbulence fractal noise to make the display feel like a movie theater screen.
